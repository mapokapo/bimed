validator: { _handleError: (e: Error) => void; _validateScale: (scale: number) => boolean; _validateWidth: (width: number) => boolean; cliDecode: (file: string) => void; cliEncode: (binary: string, file: string, overwrite: undefined | boolean) => void; decode: (values: number[], options?: DecodeOptions) => void; encode: (binary: string, options?: EncodeOptions) => void } = ...
  • Defined in utils.ts:101

An object containing validation methods for the main transform methods and the CLI transform methods.

The CLI transform requires additional validation (eg. checking if a file exists) on top of the main transform validations.
