Type declaration

  • decode: (values: number[], options?: DecodeOptions) => string
      • (values: number[], options?: DecodeOptions): string

      • Decodes an array containing raw RGB data to a string containing 0s and 1s which represent the black and white pixels.

        example

        const encoded = encode("1111000011000011", { width: 2, scale: 2, inverted: true }); const decoded = decode(encoded, { width: 2, scale: 2, inverted: true }); console.log(decoded); // "1111000011000011"

        Parameters

        • values: number[]

          An array containing the raw RGB data

        • Optional options: DecodeOptions

          The decoding options.

        Returns string

        A string containing bits that represent white or black pixels.

  • encode: (binary: string, options?: EncodeOptions) => number[]
      • (binary: string, options?: EncodeOptions): number[]

      • Encodes a string containing bits to an array of raw RGB values.

        example

        const encoded = encode("1111000011000011", { width: 2, scale: 2, inverted: true }); console.log(encoded); // Number of bytes = 192. Each row represents 2 bits = 12 bytes per row. Height (in pixels) is number of bytes / bytes per row = 16

        Parameters

        • binary: string

          The string containing the bits.

        • Optional options: EncodeOptions

          The encoding options.

        Returns number[]

        The encoded raw RGB data.
