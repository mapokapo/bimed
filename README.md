- [typescript](https://www.typescriptlang.org/)

## Getting Started

### Installation

1. Install NPM package
   ```sh
   npm install bimed
   ```
2. Use in your project
   ```javascript
   const { encode, decode } = require("bimed").default;
   // or
   import { encode, decode } from "bimed";
   ```
